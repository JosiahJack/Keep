* Play Information *
Single Player           : Yes
Cooperative             : No
Deathmatch              : No
Difficulty Settings     : Yes, easy, medium and hard
New Sounds              : No

Monster count		: 60 / 91 / 114 (easy, medium, hard)

* How to install *
Create a new folder in your Quake folder called tfl4. Unzip
pak0.pak into that folder, and start Quake like this:

	Quake -nojoy -nocdaudio -game tfl4

Remember that there are three skill settings; if you can't
finish on skill 2 (hard), try the easier settings...
