Kothic Skyboxes ( all skyboxes August 2003 )
This is version 3 of the compilation zip of all 49 of my skyboxes for Quake 3 Arena I've made to this date, September 2004.
This zip contains individual pk3 files for each skybox with the panels in tga format, rather than jpg format as in the earlier version of this pack. All the panels are 512x512.
Note that there is only one shader script, skies-kothic.shader, copied in all of the pk3 files. This makes the shader take up less space on your shader list. If you intend to modify the shader for the skybox you are using, please copy and paste it as a new shader and name it yourmapname.shader. In the case of modifications to the light information, this will only produce a difference when your map is compiled, rather than when it is run in-game, but it is good practice nonetheless.

Installation
Quake 3 Engine Games

Unzip to your baseq3 folder, or the equivelant base folder for your Quake3 engine game.
Open up the file shaderlist.txt and insert the line:

skies-kothic

The folder structure of the Quake3 engine games varies slightly for each; for Q3A itself, the shaderlist.txt is located in baseq3/scripts. Find the equivelant folder for you chosen game.
You can also extract the contents of the pk3 files to your baseq3 folder ( use Winzip; make sure the Use folder names box is checked ) to gain access to the individual components of each sky.

Quake 1

If you intend to use these skyboxes for a custom Quake 1 engine, the conventional folder location is id1/gfx/env
These skies have been released in tga format since this format is the most universally compatible with Quake engines. It is also a non-compressed image format, so if you need to convert the skybox images for use in another engine, you shouldn't lose any image quality.
I do not have any intention to release these skies in another format.

Construction
These skyboxes were made using Terragen by Matt Fairclough. It is an excellent and enjoyable program which I strongly recommend trying if you want to start making your own skyboxes.
Download the unregistered version from www.planetside.co.uk/terragen
I wrote a tutorial detailing the steps to follow to convert your Terragen world into a Quake 3 skybox; read it at www.signsofkoth.co.uk/tutorials/tut-kellskies-1.html

Post rendering manipulation was done in Paint Shop Pro - www.jascsoftware.com
