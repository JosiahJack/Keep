Changelog
Gyro 2.1a

Version 2.1a (Apr '07)
Fix: Feedback from instant forces no longer framerate dependent
Fix: Monsters no longer bounce when made into physics objects
Fix: Turbulence less prone to repeatedly hitting walls
Fix: Corrected major bug with the global turbulence subsystem
Fix: Engine power function now works correctly
Fix: Aerodynamics subsystem no longer spins objects by 360 degrees
Alteration: Minor improvement when forces lifting objects off the ground
Alteration: Forces will now affect their attached host object
Alteration: Attached forces are removed along with their attached host entity
Alteration: Area falloff functions have been renamed to allow new falloff types
Alteration: Object turbulence application now accepts an extra smoothness variable
Alteration: Global turbulence application now accepts an extra smoothness variable
Alteration: Bouyancy, resistance and turbulence mod values can now be found in gyro_user.qc
Addition: Forces may be assigned a single object to ignore
Addition: Two new force properties, ground-pound and persistence, have been added
Addition: Implemented a new cone/tube-shaped falloff component
